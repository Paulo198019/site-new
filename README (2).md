
# Publicando no GitHub Pages

## Passos:

1. Faça upload dos arquivos `index.html`, `style.css` e `banner_optimized.jpg` no seu repositório GitHub.
2. No GitHub, vá em **Settings** > **Pages**.
3. Em **Source**, selecione a branch principal (geralmente `main`) e a pasta `/root`.
4. Clique em **Save**.
5. Aguarde alguns minutos e acesse seu site pelo link fornecido pelo GitHub Pages.

### Estrutura do HTML:

```html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Debora Kulczar Advocacia</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="hero-section">
        <div>
            <h1>Debora Kulczar Advocacia</h1>
            <p>Oferecemos soluções jurídicas especializadas em diversas áreas do Direito</p>
        </div>
    </div>
</body>
</html>
```
