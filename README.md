# Débora Kulczar Advocacia - Site Institucional

Este projeto é um site institucional desenvolvido para apresentar informações sobre o escritório **Débora Kulczar Advocacia**, suas áreas de atuação, equipe e formas de contato. O design é responsivo, moderno e inclui animações suaves para melhor experiência do usuário.

---

## ✅ Estrutura do Site

- **Banner Principal**  
  Imagem do escritório como plano de fundo com sobreposição semitransparente e texto sobreposto:
  - *Excelência em Advocacia e Consultoria Jurídica*
  - Áreas de atuação destacadas e mensagem institucional.

- **Quem Somos e Sobre**  
  Seção com layout em duas colunas:
  - Imagem à esquerda.
  - Texto à direita com informações sobre a missão e valores do escritório.

- **Áreas de Atuação**  
  Cards organizados em duas linhas:
  - Primeira linha: 3 áreas (Família, Trabalho, Previdenciário).
  - Segunda linha: 2 áreas (Consumidor, Civil).
  Cada card possui ícone, título e descrição.

- **Advogada Responsável**  
  Seção centralizada com título, subtítulo e imagem da profissional responsável.

- **Rodapé**  
  Informações de contato, redes sociais e botão flutuante do WhatsApp para atendimento rápido.

---

## ✅ Tecnologias Utilizadas
- **HTML5** e **CSS3**
- **Font Awesome** para ícones
- Design responsivo com **media queries**
- Animações CSS para transições suaves

## ✅ Como Executar Localmente
1. Baixe ou clone este repositório:
   ```bash
git clone https://github.com/seuusuario/seurepositorio.git
```
2. Extraia os arquivos (se baixou o ZIP).
3. Abra o arquivo `index.html` no navegador.

---

## ✅ Publicação no GitHub Pages
1. Suba os arquivos para o seu repositório.
2. Vá em **Configurações → Pages → Branch principal → /root**.
3. O site estará disponível no link gerado pelo GitHub Pages.
