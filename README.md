# Site Débora Kulczar Advocacia

## Estrutura
- Banner horizontal 16:9 com fundo azul escuro translúcido.
- Lado esquerdo: Nome, chamada e botões.
- Lado direito: Imagem profissional.
- Seção Áreas de Atuação com cards responsivos.

## Como usar
1. Adicione a imagem `image (3).png` na pasta do projeto.
2. Substitua os arquivos no GitHub.
3. Publique no GitHub Pages.

## Personalização
- Paleta azul: ajuste no `style.css`.
- Textos: edite no `index.html`.
